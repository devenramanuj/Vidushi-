સહાય અને સમસ્યા નિવારણ

સામાન્ય સમસ્યાઓ:

❌ વૉઇસ કામ નથી કરતી:
→ બ્રાઉઝરમાં માઇક્રોફોન પરવાનગી આપો
→ HTTPS ચેક કરો (GitHub Pages પર આપમેળે છે)
→ Chrome/Edge બ્રાઉઝર વાપરો
❌ API કનેક્શન નિષ્ફળ:
→ API Key ચેક કરો (ફરીથી બનાવો જો જરૂરી હોય)
→ મોડેલ નામ ચેક કરો (gemini-1.5-flash વાપરો)
→ ફ્રી કોટા ખતમ ન હોય તે ચેક કરો (60 requests/મિનિટ)
❌ અવાજ બોલતો નથી:
→ "જવાબ આપમેળે બોલો" ચેકબોક્સ ચેક કરો
→ બ્રાઉઝર Text-to-Speech સપોર્ટ કરે છે તે ચેક કરો
→ અવાજની ગતિ 0.5x થી 2x વચ્ચે સેટ કરો

વારંવાર પૂછાતા પ્રશ્નો:

Q: API Key ક્યાંથી મેળવવી?
A: https://aistudio.google.com/apikey પરથી ફ્રીમાં
Q: GitHub પર કેવી રીતે deploy કરવું?
A: આ ફાઈલને index.html નામે upload કરો → GitHub Pages એનેબલ કરો
Q: ગુજરાતીમાં કેમ નથી બોલતું?
A: Settings → Language → ગુજરાતી પસંદ કરો
Q: મોબાઈલ પર કામ કરશે?
A: હા, બધા મોડર્ન બ્રાઉઝર્સમાં કામ કરશે
